# Fail2Ban-Report
Simple Report Tool for Fail2ban on your Webspace

# How 2 Set Up manual

+ put the .sh File on any place you like in your filesystem, let it run via daily cronjob
+ change the folder for output of the .json file to your needs
+ make a folder on your Webspace eg Fail2Ban
+ put the .php file there
+ inside of the main-folder (Fail2Ban), make the folder archive to store the json files from the .ssh script there
